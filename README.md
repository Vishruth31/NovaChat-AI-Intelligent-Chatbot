NovaChat AI – Intelligent Chatbot

NovaChat AI is a desktop-based AI chatbot application developed using Java Swing and integrated with the Gemini API to deliver intelligent conversational responses. The application features a user-friendly graphical interface designed to provide a smooth and interactive chat experience.
It incorporates persistent chat storage using MySQL, enabling efficient management and retrieval of conversation history. Additional functionalities such as message deletion and a typing indicator improve usability and create a more realistic chat environment.
This project highlights skills in API integration, desktop application development, database management, and user interface design.

Key Features

AI-powered chatbot using Gemini API
Real-time chat interface
Persistent chat storage with MySQL
Message deletion functionality
Typing indicator for enhanced user experience
Built using Java Swing for desktop environments
